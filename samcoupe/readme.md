# Information
SAM Coupe core ported from [MiST Core](https://github.com/sorgelig/SAMCoupe_MIST)
Core Id: SAMCOUPE

# How to run
A SAMCOUPE.ROM file is needed on the root of the SD-Card.

# Keys

- F11 - NMI key
- Ctrl-F11 - reset.
- Alt-F11 - reset and unload disk images.
- F12 - show OSD
